# Datalayer Provider

To get started with Notebooks hosted on [Datalayer](https://datalayer.app):

- 🚀 Create a Notebook and assign a Runtime, refer to [the official Datalayer documentation](https://docs.datalayer.app/editors-notebook/).
- 🛠️ [Install](/run) the Jupyter MCP Server on your machine.
- ⚙️ Click the `MCP Server` menu, copy-paste the commands and run it in your terminal to start the Jupyter MCP Server and connect to the Room and Runtime.

You can now use your favorite [MCP client](/clients) to interact with the Datalayer Notebook!
