# Setup

Navigate to the relevant section based on your needs:
- ***Jupyter Notebooks***: If you want to interact with notebooks in JupyterLab.
- ***Datalayer Notebooks***: If you want to interact with notebooks hosted on [Datalayer](https://datalayer.ai).
- ***Local MCP***: If you want to run the MCP server locally on your machine.
- ***Remote MCP***: If you want to run the MCP server on a remote server.
