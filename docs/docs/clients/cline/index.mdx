# Cline

![](https://assets.datalayer.tech/jupyter-mcp/cline.png)

## Install Cline VS Code extension

Install the Cline VS Code extension from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev).

## Configure Jupyter MCP Server

To use Jupyter MCP Server with Cline, add the [Jupyter MCP Server configuration](/setup/jupyter/stdio#2-setup-jupyter-mcp-server) to your `.cline_mcp_settings.json` file, read more on the [Cline documentation](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev).
